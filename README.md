# 🐱 猫の屈伸 - Pomodoro Timer

猫と一緒に集中できるポモドーロタイマーアプリです。  
集中と休憩を自動で切り替え、日別の学習時間を記録・可視化します。
---

🚀 Live Demo  
https://cat-pomodoro-1.netlify.app/


---

## 🔥 アプリ概要

「勉強時間を見える化したい」という思いから制作しました。  
シンプルなUIながら、統計機能まで実装しています。

---

## ✨ 主な機能

- ⏱ ポモドーロタイマー（作業 / 休憩 自動切替）
- ⏸ 一時停止 / リセット機能
- ⚙ 作業時間・休憩時間の設定変更（モーダルUI）
- 🐱 モードに応じた猫画像切替
- 🕒 現在時刻表示
- 📊 日付ごとの学習時間保存（localStorage）
- 🧮 総合計学習時間表示

---

## 🛠 使用技術

- HTML
- CSS
- JavaScript（Vanilla JS）
- localStorage

---

## 💡 工夫した点

- モーダルUIで設定を変更できる設計
- 日付ごとのデータ管理（オブジェクト形式で保存）

- シンプルながら拡張可能な構造
