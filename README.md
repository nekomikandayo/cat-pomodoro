# 🐱 猫の屈伸 - Pomodoro Timer

猫と一緒に集中できるポモドーロタイマーアプリです。  
集中と休憩を自動で切り替え、日別の学習時間を記録・可視化します。
---

🚀 Live Demo  
https://cat-pomodoro-1.netlify.app/


---

1. 🎯 目的（Why）

アルバイトなどで勉強時間が限られている大学生が、
短時間でも集中して学習し、その成果を可視化できる環境を提供する。

単なるタイマーではなく、

集中の継続

勉強時間の可視化

自己管理能力の向上

を目的とした学習支援アプリ。

2. 👤 想定ユーザー

アルバイトと両立している大学生

勉強時間を「見える化」したい人

継続が苦手な人

ポモドーロを使っているが記録していない人

3. 📌 機能要件
✅ MVP（最小機能）

25分作業タイマー

5分休憩タイマー

セッション回数表示

今日の合計学習時間表示

スタート / ストップ / リセット

🚀 追加機能

日別履歴保存（LocalStorage）

週間・月間グラフ表示

目標時間設定

タスク紐付け

集中度自己評価入力

ダークモード

4. 🖥 画面設計
① ホーム画面

中央：タイマー表示

下部：スタート / ストップボタン

上部：本日の累計時間

セッション数表示

② 履歴画面

日付ごとの学習時間一覧

グラフ表示（週・月）

③ 設定画面

作業時間変更

休憩時間変更

目標時間設定

5. 🗄 データ設計
Session {
  id: string
  date: string
  workDuration: number
  breakDuration: number
  completed: boolean
  taskName: string
  focusLevel: number
}

保存方法

MVP：LocalStorage

将来的にはFirebaseなどクラウドDBを検討

6. 🛠 技術選定

React：状態管理が必要なため

useState / useEffect：タイマー制御

LocalStorage：軽量なデータ保存

Chart.js：学習時間の可視化

7. 🔍 設計思想

本アプリは「集中時間」だけでなく
**“成果の蓄積と成長の可視化”**を重視している。

単発利用ではなく、継続利用を前提とした設計。
